# MB — [QA Automation] Coding Challenge

QA automation framework for the MultiBank trading platform.

## Quick Start (for reviewers)

The project is in the **`qa-automation-github`** folder:

```bash
cd qa-automation-github
npm install
npx playwright install chromium
npm run test:unit
npm run test -- --project=chromium
```

See **[qa-automation-github/README.md](qa-automation-github/README.md)** for full documentation.
